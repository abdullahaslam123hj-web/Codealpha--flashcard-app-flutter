# Flashcard Learning App

This project is developed using Flutter and Dart as part of the CodeAlpha Flutter Internship.

The Flashcard Learning App helps users learn and memorize concepts using interactive flashcards.

## Features

• Simple flashcard interface  
• Tap to reveal answer  
• Clean and user-friendly UI  

## Technologies Used

- Flutter
- Dart
- GitHub
